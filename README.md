A full stack food ordering web application where users can browse food items, add them to a cart, and place orders with online payment support.

## Features
- User authentication (Sign up & Login)
- Browse food items and add to cart
- Shopping cart and order placement
- Online payment integration using Stripe
- Admin panel to manage food items and orders
- Order status tracking

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Payment Gateway: Stripe

## Status
Currently under active development.
